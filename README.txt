What to do:
--------
On Linux
--------

Make virtual environment
> python -m venv [env_name, eg:audrecog_env]
> source audrecog_env/bin/activate

Use python 2.7
> pip install SpeechRecognition

> apt-get install python3-pyaudio python-pyaudio
> pip install pyaudio

Running speech recognition on cli:
> python2 -m speech_recognition

Running guessing games:
> python2 main.py
